# Как накатить?
## Ubuntu, Debian, Mint:
1. Ставим Python 3
2. Клонируем ветку.
3. chmod +x helium
4. ./helium
## Windows:
1. Ставим Python 3.7;
2. Качаем release, либо master;
3. Запускаем helium.cmd, либо из cmd или powershell пишем: python core.py;
4. Готово.
